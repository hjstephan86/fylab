# FyLab

Umfassendes Python-GUI-Framework mit PyQt6 für Finanzverwaltung, Portfolio-Optimierung,
Risikoanalyse und graphbasierte Finanzalgorithmen.

## Dokumentation

Das vollständige **Benutzerhandbuch** ist unter **[→ doc/MANUAL.md](doc/MANUAL.md)** verfügbar.
Es enthält:

- Architekturübersicht und Modulbeschreibungen
- API-Referenz für alle fünf Pakete (`core`, `algorithms`, `analysis`, `visualization`, `gui`)
- Theoretischen Hintergrund: algorithmische Reduktionen (Subgraph-ISO, Bellman-Ford, Kruskal, Edmonds-Karp, Markowitz)
- 13 Matplotlib-Diagramme (PNG / SVG / PDF) unter `doc/figures/`
- 4 vollständige, lauffähige Python-Beispiele

## Installation & Start

```bash
pip install -e ".[dev]"

# Tests
pytest

# GUI starten
fylab-gui
```
